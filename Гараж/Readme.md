Цикл приложения:
1.  На консоль выводится список услуг
2   С консоли пользователь выбирает услугу (в роли клиента)
3.  Запрос отправляется всем гаражам, гаражи возвращают свою цену
4.  Администратор выдает рекомендацию гаража(на основании цены)
5.  С консоли пользователь выбирает гараж (в роли клиента)
6.  С консоли пользователь подтверждает оплату (в роли клиента)
7.  Счет администратора пополняется на 10% от цены услуги
   
Рекомендации:
все сущности создавать в классе Application
